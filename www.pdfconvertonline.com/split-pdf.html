Split PDF

Split a PDF file to multiple PDF files.

Sometimes you may need to extract a few pages from a PDF file or remove some PDF pages that you don't need any more.

You can split PDF by our online program or PDFConvert desktop software for Windows.

Online

Use this form to extract selected pages or delete selected pages from PDF file.

1. Click "Choose File" button (different web browser may have different button name such as "browse..."), a browse window will open, select a local Adobe PDF file and click "Open" button.
2. Choose "Extract PDF pages" or "Delete PDF pages" options, set start page and end page, then click "Split Now!" button to split. Wait a few seconds for the task to finish.
3. You can download or view the output PDF file on your web browser when task is complete. No email address required to receive files.

Soft

Split PDF software

Split PDF by software for Windows.

After installing PDFConvert software, click Start - Programs - PDFConvert - PDFConvert to run.

Click "PDF Tools" Toolbar, Click "Split PDF" button on the left, click "Add Files" button. Browse windows will be opened, choose local PDF files (*.pdf) and click Open. When selecting a PDF file, PDFConvert will check the PDF file's security information. PDFConvert will automatically remove owner password and restrictions if the PDF file is secured by owner password. PDFConvert will prompt to enter user password if the PDF file is secured by user password which is unable to crack.

Select "Split every [page number] pages to one file." or "Extract pages from [start page number] to [end page number]" options.

Click on Convert button. PDFConvert will split the PDF files you selected and open the target folder after splitting.
