Question: How to differentiate between normal PDF and scanned PDF?

Answer: There is a simple way to confirm whether a PDF file is scanned PDF. You can open a PDF file with PDF reader such as Foxit Reader and select the text. If the text can be selected, it is a normal PDF file. If not, it is a scanned PDF file.

PDFConvert can extract text from a normal PDF file while it can't extract text from a scanned PDF file.
